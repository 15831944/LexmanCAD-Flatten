Description

Want to work with a 2D polyline but for some reason your drawing contains a 3D polyline instead?

Applying the "LCAD_FLATTEN" command to a 3D polyline will replace a 3D polyline with a 2D polyline.

The X and Y geometry will be preserved and the resulting 2D polyline will be placed at a user-specified elevation.

If you need to convert several 3D polylines within a drawing, multiple 3D polylines can be selected and "Flattened" at once.

From version 2.0 onwards, the reverse is also possible. Using the command "LCAD_RAISE" 2D polylines can be converted into 3D polylines.

General Usage Instructions

- Use the ribbon buttons or the commands "LCAD_FLATTEN" and "LCAD_RAISE" to convert 3D Polylines into 2D Polylines and vice versa.

- Select one or several 3D polylines to flatten, or one or several 2D Polylines to raise.

- In the case of "Flatten," specify an elevation to place the flattened 2D Polyline. In the case of "Raise," the each point of the resulting 3D Polyline will be placed at the elevation of the 2D Polyline which is being converted.

-"Flatten" will replace 3D Polylines with 2D Polylines while "RAISE" will replace 2D Polylines with 3D Polylines.

Commands

Ribbon/Toolbar Icon Command Command Description

LCAD_FLATTEN

- Launching LCAD_FLATTEN will prompt the user to select 1 or multiple 3D polylines. - After selection, hit enter to continue (or hit escape to exit the command). - User is then prompted to specify a "target elevation". Target elevation is the elevation at which the 2D polyline will be placed. A numeric entry is required and the default entry is zero elevation. Hit enter to continue. - The 3D polyline is then replaced with a 2D polyline at the specified elevation.

LCAD_RAISE

- Launching LCAD_RAISE will prompt the user to select 1 or multiple 2D polylines. - After selection, hit enter to continue (or hit escape to exit the command). - The 2D polyline is then replaced with a 3D polyline.

Installation/Uninstallation

The installer that ran when you downloaded this plug-in from Autodesk Exchange Apps has already installed the plug-in. You may need to restart the Autodesk product to activate the plug-in. To uninstall this plug-in, click Control Panel > Programs > Programs and Features (Windows 7/8/8.1) or Control Panel > Add or Remove Programs (Windows XP), and uninstall as you would any other application from your system.

Additional Information

"Flattening" a 3D poly to a specific elevation and then "raising" it again is a quick way to get a 3D poly with all points at the same, user-specified, elevation and is much quicker than moving the points manually.
